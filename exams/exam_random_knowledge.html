Random Knowledge Sampling Exam

Basics

Bits Manipulation

  1. Count the number of “1” bits in a char.

    • How would you do this in O(1)?

  2. Write a function that reverses the order of bits in a char.

    • How would you do this in O(1)?
    • How would you do this in O(1) for an unsigned int?

  3. Calculate in O(1) if a number is a power of 2.

  4. Write a function which implements shift rotate right it 1. Should get a number and how much shift to do.

Recursion and Iteration

Perform the following iteratively and recursively:

  1. Flip a linked list.

  2. Print the content of a BST.

  3. Fibonacci(n):

    • What is the preferred way (recursion / iteration)?
    • How can you improve Fibonacci recursion?

  4. Sort a stack.

Miscellaneous

  1. Detail the compilation process.

  2. Implement memset / memcpy.

  3. List 3 sorting methods and their complexities.

  4. Draw the structure of a program’s memory segments.

    • What can be found in each segment?

DS

  1. Specify the usage and complexity of a hash table.

    • What problems is it used to solve?

  2. Detect if there is a loop in a linked list, and open it.

  3. You are reading a data stream. How would you maintain a 1. list of the last 5 elements you have read? (What would you use instead of a list?)

  4. List 5 data structures. What is the complexity O(?) of the operations for each structure?

  5. How would you implement LRU cache? (Find should be O(1), LRU 1 and Eviction should be O(1)).

OS

  1. Define the following:

    • Page
    • Page fault
    • Interrupt (soft and hard)
    • Segmentation fault
  2. Describe everything that happens between pressing a key on the keyboard and the appearance of a letter on your screen.
  3. Write pseudo code for a multi-threaded, fixed-size queue.
  4. Detail the differences between a process and a thread.

  5. Detail the differences between a mutex, semaphore, and condition variable.

  6. What can cause a context switch?

  7. List 10 ways of passing information between processes.

  8. How would you pass information between threads?

  9. In 8 words or less, how would you prevent deadlock?

  10. A function foo() does i++ on a global variable that is initialized to 0. Two threads are calling the function, five times each. What are the minimal and the maximal possible values of i in the end?

  11. What are the differences between TCP and UDP?

  12. What are the best alternatives between interrupt and 1. polling mechanism, and why?

  13. Write pseudo code for a thread-safe producer / consumer (with multiple producers, multiple consumers).
